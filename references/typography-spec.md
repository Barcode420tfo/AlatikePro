# Typography Spec

## Font Pairing

- Display / headlines: `Playfair Display`
- Body / UI / labels: `Lato`

## Source

Both fonts are loaded from Google Fonts and are free to use.

## Usage

- `Playfair Display` carries the luxury editorial tone for hero copy, section titles, and other high-emphasis text.
- `Lato` handles paragraphs, navigation, buttons, cards, labels, and general interface text.

## Weights

- `Playfair Display`: `400`, `500`, `600`, `700`
- `Lato`: `300`, `400`, `700`
