# Alatike Pro

React + Vite + Tailwind build for the Alatike Pro beauty website.

## Current Focus

- Clean project scaffold
- Navbar
- Hero section
- Reference-led motion and hover behavior

## Project Structure

```text
.
├── assets
├── media
│   ├── photos
│   └── videos
├── references
├── src
│   ├── components
│   │   └── layout
│   ├── data
│   ├── sections
│   │   └── hero
│   └── styles
└── index.html
```

## Notes

- Brand assets and client media stay outside `src` so they remain easy to manage.
- Page code lives inside `src`.
- We are building the site section by section, starting with the navbar and hero.
